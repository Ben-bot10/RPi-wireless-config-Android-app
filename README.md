# RPi-wireless-config-Android-app
An Android app to wirelessly transmit Wi-Fi credentials (SSID and Password PSK) to a Raspberry Pi via Bluetooth, helps in headless setup of Raspberry Pi devices, eliminating the need for a keyboard, mouse, or monitor during initial configuration.

## Credits
This project is based on [rpi3-wifi-conf-android](https://github.com/brendan-myers/rpi3-wifi-conf-android)

Tested on Raspberry Pi 4 running dietpi.

Use it with the script run.py on Rpi

- Made it compatible with modern Android versions.
